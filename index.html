What is a Future

aka Promise

A value that may or may not be yet available

Use-Case: Network-Requests


							main thread     OS/network hardware
							fetch URL ---------->
							...                  |
							...                  |
							...                  |
							use fetched data <---
  • main thread can do other work in the meantime

Use-Case: Computation


							main thread     computation threadpool
							start computation -->
							...                  |
							...                  |
							...                  |
							use result <---------
  • main thread doesn't block (e.g. GUI thread)
  • threadpool can be saturated with work
